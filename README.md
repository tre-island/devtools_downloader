# scoop_scripts
Windows package manager install scripts

This is a repository used to the scoop windows package manager to install common tools and software

Using these scripts can help to quickly get a development environment up and running (especially virtualmachines).

## Requirements
- Windows OS
- Powershell 5.0 or above
- .NET Framework 4.5 or aboves

Scripts in the repository are intended to be lightweight, with minimual error checking.
